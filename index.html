CSS

CSS is a crucial part of HTML. Basically all the design work is done on that sheet. You can do things like include different types, classes, or IDs, change background and font colors, and resize articles/sections to fit your needs just to name a few.

Section IDs

Something that's pretty cool to web development is that in HTML you can name different articles or sections to be able to have custom sets of rules applied to them. That way you can have the same or different size fonts, padding, margins, etc. for various sections within one page of the site or on all. This technique is being used currently on this website to specify a consistent width between them all.

Pre-Production

I learned about the importance of pre-production when it comes to designing and developing a website. Going through ahead of time and constructing a layout for each page will save time when it comes to writing the actual code for it. Having something to go off of is much easier than trying to code with no wireframes, comps, etc.
